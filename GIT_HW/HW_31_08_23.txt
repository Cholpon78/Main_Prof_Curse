Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?     
        1 шаг: Надо проверить настройки имени и почты. git config --global -l
        Если имя и почта не установлдены надо установить: 
        git config --global user.name "Имя и ФИО"
        git config --global user.email "фдрес эл-почты"
        2 шаг: Выбрать на GITHUB репозиторий который хотим скопировать. Скопировать URL aдрес (выбираем https)
        3 шаг: В GitBush переходим в нужную директорию по команде cd <путь к папке>
        4 шаг: В GitBush пишем команду git clone <скопированный адрес URL>


Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
1 - локально переходим в нужную директорию через команду cd <путь к папке>
2 - ввести команду git init. В директории появится скрытый файл .git
3 - вводим команду git status, чтобы проверить какие файлы нужно перенести в репозиторий
4 - если переновим все файлы - git add . 
    если отдельный фаул -git add <имя файла>
5 - команда git commit - m " комментраий"
6 - на GitHub создаем новый репозиторий
7 - связываем локальную папкус репозиторием - git remote add origin <ссылка на репозиторий>
8 - команда git push

Как получить на локальный компьютер изменения с github.com
1 - перейти в локальную папку которую хотим обновить
2 - затем команда git pull  

Какую информацию нам дают команда :
git status - показывает какие файлы были изменены локально, и какие были добавлены в index
git log - можно увидеть все коммиты
git remote -v - в каком удаленном репозитории находимся
